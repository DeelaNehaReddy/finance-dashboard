## 📂 Project Structure
# 💰 Finance Dashboard UI

A simple and interactive finance dashboard built using React.  
This project focuses on clean UI design, component structure, and handling frontend data effectively.

---

## 🚀 Overview

This dashboard allows users to:

- View overall financial summary (balance, income, expenses)
- Explore transaction history
- Understand spending patterns through charts
- Experience basic role-based UI behavior

The goal of this project is not complexity, but clarity — showing how a real-world dashboard can be structured and built from scratch.

---

## 🛠️ Tech Stack

- React (Vite)
- JavaScript (ES6+)
- CSS (Custom styling)
- Recharts (for data visualization)

---

## ✨ Features

### 📊 Dashboard Overview
- Displays Total Balance, Income, and Expenses
- Line chart showing trend over time
- Pie chart showing spending breakdown

### 📋 Transactions
- List of transactions with:
  - Date
  - Amount
  - Category
  - Type (Income / Expense)
- Clean and readable table UI

### 🔐 Role-Based UI (Frontend Simulation)
- Viewer → Can only see data
- Admin → Can access additional actions (like edit placeholder)

### 📈 Insights (Basic)
- Users can visually understand:
  - Spending patterns
  - Income vs expense trends

### 📱 Responsive Design
- Works across different screen sizes
- Flexible layout using simple CSS
